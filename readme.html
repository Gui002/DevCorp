DevCorp - informação aos colaboradores do projecto

Greetings

Este diretório, o root do projecto, contém todos os recursos locais que serão usados para o DevCorp. Isso significa que todos os recursos devem ser adicionados nesta pasta nos diretórios apropriados, sendo esses:

  • Código fonte e ficheiros HTML e CSS ou similares
  • bibliotecas externas
  • Recursos de media
    • Áudio
    • Vídeo
    • Imagens
    • Fontes→ isso não é media mas whatever
    • Gifs
  • E ficheiros de configuração do site
  • Documentacao do sistema

Onde:

Código fonte
são ficheiros escritos com recurso a uma linguagem de programação
Ficheiros de configuração
são ficheiros que são que contém alguns parâmetros que sao usados pelos programas para definir aspectos que se pretende persisir na aplicação como:
  • Temas - cores de alguns paineis
  • Variáveis (parâmetros) de configuração de banco de dados
  • etc.
Documentacao do Sistema
são os documentos que dizem respeito a organizacao da estrutura e comportamentos do sistema. estes serão preenchidos conforme a necessidade. os diretórios foram criados como forma de garantir que caso haja necessidade, os locais já estejam definidos
